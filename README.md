# minwoo.kim 블로그

## 실행 방법

### 사전 실행 (Clone 후 실행)

```bash
git submodule update --init --recursive # ./scripts/init.sh
```

위 명령어를 수행하여, git submodule을 업데이트해준다.

### 로컬 실행

```bash
hugo server -D # ./scripts/run-local.sh
```

### 빌드

```bash
hugo --gc --minify # ./scripts/build.sh
```
